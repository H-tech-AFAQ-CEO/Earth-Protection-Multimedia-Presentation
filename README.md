# 🌍 Protect The Earth – Interactive Presentation

An engaging, animated, and fully interactive **web-based environmental awareness presentation** built using pure HTML, CSS, and JavaScript. Designed to educate and inspire students about sustainability, climate change, and everyday eco-actions.

---

## 🚀 Quick Start

1. Download or copy the file  
2. Open it in any modern browser:

✅ No installation, no setup — works instantly!

---

## 🎯 Purpose

This project is designed to:
- Educate students about environmental issues  
- Promote sustainable habits  
- Deliver content in an interactive and visually engaging way  
- Combine learning with animations and quizzes  

---

## ✨ Features

- 🌌 Animated particle background  
- 🌍 Interactive Earth hero section  
- 💧 Water & energy awareness slides  
- ♻️ Waste segregation visual guide  
- 🌡️ Climate change statistics  
- ⚡ Actionable eco-friendly tips  
- 💚 Interactive pledge system  
- 📊 Real-time global counters  
- 🧠 Built-in quiz system  
- 🎮 Keyboard navigation (arrow keys)  
- 📱 Fully responsive design  

---

## 🧭 Navigation

- Use **Arrow Keys (← →)** to move between slides  
- Or use on-screen navigation buttons  
- Progress bar shows your position  

---

## 📚 Slides Overview

1. **Intro / Hero** – Earth animation & mission start  
2. **Water & Energy** – Key facts and savings  
3. **Recycling** – Waste sorting system  
4. **Climate** – Global environmental data  
5. **Actions** – What YOU can do  
6. **Pledge** – Commit to eco habits  
7. **Quiz** – Test your knowledge
